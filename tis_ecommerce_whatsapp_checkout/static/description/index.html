eCommerce WhatsApp Checkout

eCommerce Checkout through WhatsApp

Configuration

Enable Whatsapp Checkout if need Whatsapp Checkout option

Configure WhatsApp Number(include country code).

Also Visibility Control of WhatsApp Checkout button.


Process WhatsApp Checkout

Click on WhatsApp Checkout button, it will redirect to your WhatsApp.


Once checkout to WhatsApp, then cart becomes empty.
